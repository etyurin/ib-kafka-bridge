# ib-kafka-bridge

Interactive Brokers API to Kafka bridge.

## Usage

```lein run --kafka-uri localhost:9092 --tws-uri localhost:7500:117 SPY EUR.USD```

## Prerequisites

 - Trader Workstation
 - Compiled Interactive Brokers API 
 - Kafka
 - Leiningen
 - JVM
